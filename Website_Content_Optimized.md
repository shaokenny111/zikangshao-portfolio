## ❻ 重要的几条整体修正建议

按对效果影响从大到小排：

1. **🔴 标题里的 "Stitch 占位人设" 必须全部清空**——FinTech Global Corp、Capital Alpha Partners、AlphaPredict、Ledger Flow 这些是 demo 假数据，留着会让招聘者瞬间失去信任。

2. **🔴 副标题换掉**（见 ❶ 区）。"Bridging the gap…" 是 AI 写出来的、谁都能用的话，没有把"你"留下来。

3. **🟡 Toolbox 完全不匹配你的背景**——现在写着 UX Research / React / Figma，会让招聘者以为你是产品设计师而不是 finance + AI。按 ❹ 区替换。

4. **🟡 顶栏中文/英文问题**——目前只有中文导航，建议英文优先（你的简历是英文、目标读者更可能是国际/双语招聘者），右上角 EN/中 切换器保留。

5. **🟢 Featured Project 视觉差异化**——`AI Research Engine` 是你的最强卖点，建议在作品集里做"占整行宽度"的特殊版式，区别于其他 3 张普通卡。已经在原 PRD §5.4 写过规格。

6. **🟢 Background 三个 tab 的默认 tab**——建议默认打开 Work Experience（最新工作 = 招聘者最关心）。Stitch 现在已经做对了 ✅。

7. **🟢 Click to Expand 一致性**——设计稿现在只有第一张卡有 expand，其他没有。要么全部加，要么全部不加。建议**全部加**——折叠态保持页面短小，详情按需展开。
