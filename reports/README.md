# Reports

This folder stores PDF and screenshot assets for the case study pages.

## Files to upload (by you):

- `ml-backtesting.pdf`  — The ML Backtesting full report (374 KB)
- `guoyuan/01.png` ~ `05.png`  — Guoyuan auto research sample pages
- `take-two/01.png` ~ `05.png`  — Take-Two research sample pages
- `robotics/01.png` ~ `05.png`  — Robotics research sample pages

## Image format:

- PNG, long edge ≥ 1600px
- Each file < 500 KB (compress with tinypng.com)
- Sequential naming starting from `01`
- The carousel auto-discovers up to 12 sequential images (stops at first missing number)

## Filename rules:

- All lowercase
- Two-digit zero-padded numbering: `01.png`, `02.png`, etc.
- The viewer scans `01` → `12` and stops at the first missing file
